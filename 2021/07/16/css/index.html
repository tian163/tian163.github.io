2.css选择器

作用：选择页面上的某一个或者某一类元素

优先级：不遵循就近原则，id选择器>类选择器>标签选择器

2.1基本选择器

​ 1.标签选择器，会选择到页面上所有的这个标签元素

​ 2.类选择器 class，可以多个标签归类，是同一个class，跨标签

​ 格式：

​ .class的名称{

​ }

​ 3.Id选择器,id必须保证全局唯一

​ 格式：

​ #id名称{

​ }

2.2层次选择器

​ 1.后代选择器：在某个元素的后面

​ 2.子选择器:一代,儿子

​ 3.相邻兄弟选择器:同辈,只有一个，相邻（向下）

​ 4.通用兄弟选择器：当前选中元素向下的所有兄弟

2.3.结构伪类选择器

伪类：行为

a:hover //鼠标悬浮的颜色

a:active //激活：鼠标按住未释放的状态

2.4属性选择器(常用)

id+class结合

属性名，属性名 = 属性值(正则)

6.浮动

  1. 1块级元素(block)：独占一行

​ h1~h6 p div 列表

​ 行内元素(inline)：不独占一行

​ span a img strong

​ inline-block:保留块级元素特性但可以写在一行

​ 行内元素可以包含在块级元素中，反之则不行

​ 6.2display

​ float左右浮动

​ 6.3父级边框塌陷问题

​ clear：right；右侧不允许有浮动元素

​ clear：left； 左侧不允许有浮动元素

​ clear：both； 两侧都不允许有浮动元素

​ clear：none；

​ 解决方案：

​ 1.增加父级元素的高度

​ 2.增加一个空的div标签，清除浮动

​ 3.在父级元素中增加一个overflow：hidden；

​ 4.父类添加一个伪类 ：after

7.定位

​ 7.1相对定位

​ 1.没有父级元素的基于浏览器定位

​ 2.假设父级元素存在定位，我们通常会相对于父元素进行定位

​ 7.3 固定定位

​ position：fixed；
